Obsługa eventów

Chcemy, żeby nasze strony reagowały na użytkownika, żeby wyświetlały mu informacje zaraz po tym, jak on coś kliknie lub wpisze.

po to używamy JavaScriptu, żeby nasze strony były INTERAKTYWNE.

3 kroki

  • wybierz element
  • wybierz zdarzenie
  • zakoduj, co ma się stać

Jakie zdarzenia możemy wybrać

  • load
  • resize
  • keydown/keyup/keypress
  • click
  • mouseover
  • focus
  • input
  • change
  • submit

Jak to robimy? 

					
					var element = document.getElementById("button_first");
var doIt = function(){ alert("Kliknięte")};
element.addEventListener( 'click', doIt, false);
} );
Co tu jest nietypowego?

A jak jest w jQuery?

Oczywiście łatwiej... 

					
					$('#button-first').on('click', function(){
	alert("Ojej!!")
})

Jak wykorzystać sam event w funkcji? 

					
					var element = document.getElementById("button_first");
var doIt = function(e, name){ alert("Kliknięte"+name);
console.log(e)};
element.addEventListener( 'click', function(e){doIt(e, "Kasia")}, false);
} );
